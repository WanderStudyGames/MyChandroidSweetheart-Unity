start string with  `$`
```
<size=22> Big text </size>
<color=green> Healed player </color>
<color=#ff0000> Specific color </color>
<b> Bold Text </b>
<i> Italics </i>
```

Find common colors [here](https://docs.unity3d.com/Packages/com.unity.ugui@1.0/manual/StyledText.html#supported-colors)
